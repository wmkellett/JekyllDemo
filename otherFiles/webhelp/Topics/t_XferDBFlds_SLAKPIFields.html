SLA/KPI Fields

This table contains low-level information regarding each CHECK command run (that is, each checkpoint).  Note that only CHECK commands that contain the ConditionsMet parameter are recorded in the VLTrader enterprise database.

VLSLAKPI database table
Column Name Data Type Length Description
TransferID VARCHAR 30 TransferID in VLTransfers table
VLSerial VARCHAR 6 VLSerial in VLTransfers table
CheckType VARCHAR 30

Contains one of these values:

  • File Age
  • Directory Age
  • Transfer
  • Transfer EDI Ack
  • Transfer Response
Class VARCHAR 20

Contains one of these values:

  • SLA
  • KPI
  • BLANK
Subclass VARCHAR 30

Contains one of these values:

  • user-specified string
  • BLANK
TimeCondition VARCHAR 100 Contains a common-language summary of the primary time condition.
Direction VARCHAR 20

Contains one of these values:

  • Inbound
  • Outbound
  • ‘n/a’
Recursive VARCHAR 10

Contains one of these values:

  • True
  • False
  • ‘n/a’
Status VARCHAR 20

Contains one of these values:

  • Delivered
  • Completed
  • Any
  • ‘n/a’
Count VARCHAR 20

Contains one of these values:

  • 1-99999
  • All
  • ‘n/a’
Sender VARCHAR 500

Contains one of these values:

  • user-specified string
  • BLANK
  • ‘n/a’
Receiver VARCHAR 500

Contains one of these values:

  • user-specified string
  • BLANK
  • ‘n/a’
FunctionalGroupSender VARCHAR 500

Contains one of these values:

  • user-specified string
  • BLANK
  • ‘n/a’
FunctionalGroupReceiver VARCHAR 500

Contains one of these values:

  • user-specified string
  • BLANK
  • ‘n/a’
TransactionType VARCHAR 6

Contains one of these values:

  • user-specified string
  • BLANK
  • ‘n/a’
DocumentType VARCHAR 255

Contains one of these values:

  • user-specified string (planned)
  • BLANK
  • ‘n/a’ (planned)
ResponseMatchCondition VARCHAR 100

Contains one of these values:

  • user-specified string (planned)
  • ‘n/a’ (planned)
ConditionsMetClassification VARCHAR 20

Contains one of these values:

  • Success
  • Error
ConditionsMet BIT   ‘true’ if overall conditions of CHECK were met; ‘false’ otherwise
