EBICS Mailbox: Security Tab

Since it is mandated for EBICS, HTTP/s is pre-selected on the Security tab. With HTTP/s, the target bank server can issue client certificates. If so, import the client certificate using Certificate Management and then specify (or browse to) the imported Certificate Alias and specify a Password.
