Transfer Status Filter

The Transfer Status Filter panel allows you to select which items you wish to view. By default, it will display the current day's transfers. The From and To fields allow you to select the range of times you wish to view. You may either use the pull-down menus or enter dates (YYYY/MM/DD) or times (HH:MM) into the fields. There are four possible tabs that might be present on this panel:  General, Tracking, VersaLexes, and Advanced.

Save As… can be used to save the current filter settings in an XML file.  You can then use this XML file to later recall these settings using Open… or with the VLStatc -f command-line option to use the filter for reports generated to a file or to an email address.

After selecting the appropriate filter, select Continue to read the data and display the data in the transfer report.
