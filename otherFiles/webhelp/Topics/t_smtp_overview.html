SMTP Overview

The generic SMTP and SMTP/s hosts are provided to allow a user to fully specify a client email interface to an SMTP server. Using a pre-configured host specific to the target server, if possible, will save you the effort of having to research, specify, and then debug the interface.

Not all SMTP servers will support or require the full set of host options allowed by VersaLex. The following action commands are available on VersaLex:
