Launching Cleo Dashboards

Before you can access the Cleo Dashboards™ software, your system administrator must set up the appropriate access through VersaLex and Cleo VLNavigator. Your administrator will provide an launch URL in the format:

http(s)://VersaLexComputerIP:http(s)Port/DashboardsResourcePath

where
VersaLexComputerIP
IP address of the computer where the Cleo Harmony software is installed
http(s)Port
Port number where the Cleo Harmony software is listening for HTTP or HTTPs activity
DashboardsResourcePath
The path to your Cleo Dashboards resources. Configure this property in Local Listener Web Browser properties. See Configuring Dashboards and System Monitor for Web Browser service.
  1. Enter the URL provided by your administrator in your browser.
    A login dialog box displays.
  2. Enter your username and password, and click Login. 
    The Cleo Dashboards module allows access to only one VersaLex pool at a time. If your user account is associated with multiple pools, you will be prompted to choose a pool. After selecting the desired pool, it will compare the VersaLex version against the version of the report files in the report server. If the versions of the report files do not match the VersaLex version, then the report files will be published (see Publishing Reports).
  3. The Cleo Dashboards module allows access to only one VersaLex pool at a time. If your user account is associated with multiple pools, you will be prompted to choose a pool. 
    The version of VersaLex in the pool selected is compared to the version of the report files in the report server. If the versions of the report files do not match the VersaLex version, the report files will be published (see Publishing Reports). The browser displays a window containing report information.
  4. If you need access to both Cleo Dashboards and Cleo System Monitor software at the same time in the same browser, you must create a new session for the second application and then use the appropriate URL.
    The steps to create a new session differ depending on the chosen browser.
    • For Internet Explorer, select File > New Session.
    • For Chrome, select File > New Incognito Window.
    • For Firefox, select File > New Private Window.
