Supplementary Dashboards

Besides the baseline dashboards discussed in the previous sections, there can also be supplementary dashboards available. These dashboards can be accessed via Public Reports>Business Dashboards>_Supplementary. The table below provides a brief description of the supplementary business dashboards. 

Dashboard Name Description
EDI Transactions (Inbound) Summarizes tracked data for all inbound EDI transactions. Transactions are mapped to their associated transfers.
EDI Transactions (Outbound) Summarizes tracked data for all outbound EDI transactions. Transactions are mapped to their associated transfers.
EDI Transactions Summarizes tracked data for all EDI transactions, both inbound and outbound. Transactions are mapped to their associated transfers.
Inbound Transactions Summarizes tracked data for all inbound EDI, XML, and Text transactions.
Outbound Transactions Summarizes tracked data for all outbound EDI, XML, and Text transactions.
