Configuring Local AS3 Message Headers

The AS3 tab contains the configuration for the AS3 message headers.

  1. Enter the AS3 name that you will be using for this trading relationship in the AS3-From field.
  2. Enter your trading partner’s AS3 name in the AS3-To field.
  3. Enter some ASCII text in the Subject field.  This text will be included in the header of all messages sent to this trading partner.
  4. Select the appropriate Content-Type of the files you will be sending to this trading partner from the pull-down list.  You can also leave this field blank and allow the application to detect the Content-Type first based on the file content and then the file extension. 
    Detectable types include:
    • application/edifact
    • application/edi-x12
    • application/edi-tradacoms
    • application/xml (text/xml)
    • application/pdf
    • application/msword
    • application/x-msexcel
    • application/rtf
    • application/zip
    • image/bmp
    • image/gif
    • image/tiff
    • image/jpeg
    • text/plain
    • text/html
    • video/mpg
