Transfer Table Standard View

The Transfer Report panel shows the status of the items that match the filter selected. The above screenshot shows, for the most part, the default set of transfer columns. The File Type column is displayed only if file tracking is enabled. This column will show 'EDI' (EDI-X12, EDIFACT, or TRADACOMS), ‘XML’ or ‘Text’ depending on the configured file tracking options and the file transferred. The Run Type column is only available when database transfer logging is being used. If VLNavigator user groups are employed, additional columns can be added, columns can be removed, and the order of the columns can be set per user group (refer to File Transfer Report Tab under User Group).

Initially the table is sorted by the Start Time of the transfer. By clicking on any of the column headers, the table can be sorted in ascending or descending order based on the selected column's contents.

Note that a transfer could have a status of 'Delete Error' or 'Delete Resolved'. These are special statuses associated with a monitoring feature of the Cleo Harmony application. See Source Deletion Monitoring for more information.

Push buttons at the bottom of the table can be used to change filter settings and refresh the table, find entries in the table, refresh the table with the current filter, generate an HTML report, and export the table to a CSV file.
