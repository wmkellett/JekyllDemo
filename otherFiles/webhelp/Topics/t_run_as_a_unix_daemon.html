Run as a Unix Daemon

WARNING: The following procedures have been tested with specific distributions of Solaris, HP-UX, Linux, and AIX; consult your system documentation to ensure that these steps are correct before starting. Review the run levels (rc#.d) and sequence numbers (S# and K#) given for appropriate values. Only the system administrator should perform these changes.

Note: This section contains numerous command line examples referring to VersaLex, VersaLex c, and VersaLex d.  Please substitute VersaLex with your specific product name (either LexiCom, VLTrader, or Harmony).  For example, VersaLex d in practice becomes either LexiComd, VLTraderd, or Harmonyd.
Note: Prior to installing VersaLex as a Unix daemon, the following command from the VersaLex installed directory can be used to first verify that Versalex is operational:
  ./  VersaLex c -s "service" –m
