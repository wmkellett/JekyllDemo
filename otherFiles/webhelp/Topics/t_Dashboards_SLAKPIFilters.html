SLA/KPI Dashboard Filter Area

The following sliders and filters are provided on the SLA/KPI Dashboard. These filters are associated with the library components on the dashboard; when one filter is selected, all the components will be updated accordingly.

  • Date/Time Range Slider: This slider is bound to the Start Time field and controls all components on the dashboard.
  • Trading Partner Filter: This filter control is bound to the Trading Partner field and controls all components on the dashboard.
  • Host Folder Filter: This filter control is bound to the Host Folder field and controls all components on the dashboard.
  • Check Type Filter: This filter control is bound to the Check Type field and controls all components on the dashboard.

See Filtering Dashboard Component Data for information about manipulating sliders and filter controls. The range of data values represented in the sliders/filters is comprised of all the values within the current record set. The current record set is defined as all records returned from the most recent query. For the SLA/KPI Dashboard, queries are executed at these times:

  1. When the dashboard is first launched.
  2. When Refresh is clicked on the side bar.
  3. When new parameters are submitted by clicking Enter Parameter Values on the side bar and entering new parameter values into the Enter Parameter Values dialog box(see SLA/KPI Dashboard).
