Transfer Dashboard Parameters

The Transfer Dashboard contains the following configuration parameters:

  • Start Date/Time: this parameter specifies the starting date/time for the query. The factory-supplied default value is DateAdd('h', -24, CurrentDateTime()), which translates to 24 hours ago from current date/time.
  • End Date/Time: this parameter specifies the ending date/time for the query. The factory-supplied default value is CurrentDateTime()which translates to the current date/time.

These factory-supplied parameter settings can be overridden by entering new parameter values. See Specifying Dashboard Parameter Values for more information about setting parameters.

Note: Due to practical memory limitations, if you specify a date range that produces a large number of records, the query will automatically limit the number of records.The current limit is 100,000 records; however, this value is expected to increase in the future. If the date range you specify creates a condition where the number of records has been limited, a warning message will be displayed at the bottom of the Transfer Details table. This indicates that you are not looking at a complete set of data. After receiving this warning, the date range should be decreased until the warning goes away.
