Summary Dashboard SLA/KPI Pie Chart

The SLA/KPI Pie chart summarizes the checkpoints for each trading partner bound by the current filter (see Summary Dashboard Filter Area) and parameter settings (see Summary Dashboard Parameters. The initial chart categorization is General Status. This can be changed through the Go To and Go to by Value right-click menu options. See Changing the Summary Dashboard chart and series and Changing Summary Dashboard chart, series, and filter.

These options operate the same as the Transfers pie chart. See Summary Dashboard Transfers Pie Chart for more information.
