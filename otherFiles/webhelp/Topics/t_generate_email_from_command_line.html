Generate Email from Command Line

LexMailc or VLMailc can be used from the command line to generate an email. Usage of command line LexMailc/VLMailc is as follows:

usage: LexMailc/VLMailc

-?
Displays the command line options
-a   address(es)
Comma-separated list of email recipients [REQUIRED]
-r    sender
Email address of the sender [REQUIRED]
-e   emailserver
SMTP email server to use for sending the message [OPTIONAL]
-u   username
SMTP email server authentication username [OPTIONAL]
-p   password
SMTP email server authentication password [OPTIONAL]
-s   subject
Subject of the message [OPTIONAL]

Although the following options are not required, at least one MUST be specified:

-m  textmsg
Free-form text to include in the body of the message
-t   textfile
Path of the file containing text to include in the body of the message
-f   attachment(s)
Comma-separated list of paths to files to be attached to the message
