Configuring a whitelist

A whitelist allows you to specify IP addresses from which users are allowed to log in. You specify whitelist IP addresses per local user mailbox by editing individual local user mailboxes. See Configuring IP filtering for an FTP mailbox, Configure IP Filter for Local HTTP Mailbox, Configure IP Filter for Local SSH FTP Mailbox, and Users Mailbox: IP Filter.
