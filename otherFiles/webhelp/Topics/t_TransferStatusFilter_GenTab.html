Transfer Status Filter - General Tab

In addition to specifying a date range, and any filter selections made on the other tabs, you can also filter based on specific trading partner connections, folder names, and hosts\mailboxes. The All and None buttons allow you to select all/none of the criteria for the particular section for which the button applies. If Show Mailboxes is checked, then all the mailboxes within each listed host will also be displayed. Note that transfers associated with host-based actions will only be displayed if Show Mailboxes is unchecked and the associated host is selected.

The Username(s) filter is shown if an LDAP Server is configured and enabled. If desired, enter a comma or semi-colon separated list of LDAP usernames. If this field is set, then all but local FTP, HTTP, and SSH FTP LDAP  mailboxes will be excluded and only transfers for the specified users will be included.
