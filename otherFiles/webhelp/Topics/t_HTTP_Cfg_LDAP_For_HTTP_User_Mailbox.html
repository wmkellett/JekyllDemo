Configure LDAP for Local HTTP Mailbox

If necessary, select the Override System Options check box for Base DN and Search Filter and provide appropriate values (see LDAP Server) in order to match the intended set of users for this mailbox. You can also use the Extend Search Filter field to append rules to the default system search filter.

Use List to list the users and their attributes matching the Base DN and Search Filter.
