Dashboard Editing Area

The dashboard editing area consists of the dashboard header and the dashboard body. 

The dashboard header can contain following the resources (see Resources Panel):

  • Dashboard Title
  • Image
  • Label
  • Special Field

The dashboard body can contain the following resources (see Resources Panel):

  • Library Components
  • Sliders
  • Filter Controls
  • URL Frames
  • HTML Frames
Note: It is also possible to include report components into dashboard bodies; however, doing so is not recommended, as web reports should generally be accessed directly from library components.
