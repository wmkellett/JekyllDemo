Users Overview 

When you start your FTP, HTTP, or SSH FTP server for the first time, no Users exist and therefore no access is granted to your server.  To initiate creation of Users, first activate the Users template (see Activate a Host from a Template) and then add a new user mailbox.

User mailboxes can have actions, but unlike remote host/mailbox actions that perform remote host operations, User actions can only perform local operations that manipulate files within the user's home directory. 

Users can be either native users or LDAP users.
