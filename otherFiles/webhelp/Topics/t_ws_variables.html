WS Variables

The Web Service protocol utilizes variables both as arguments to methods, and to store the results from method calls. The Cleo Harmony application also provides a series of predefined variables that can be used as arguments to methods. The predefined variables in the all section can also be used in the Custom XML Soap Headers field. SeeWS Mailbox: Headers Tab.
