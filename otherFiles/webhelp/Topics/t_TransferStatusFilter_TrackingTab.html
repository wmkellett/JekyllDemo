Transfer Status Filter - Tracking Tab

In addition to specifying a date range, and any filter selections made on the other tabs, if one of the file tracking features is enabled (refer to Transfers under Configure System Options) you can also filter based on certain tracking criteria. You have the option to filter by either EDI, XML or text data (the screen above shows the available filters for EDI). For the Reference 1 and Reference 2 filters, wild-carded searches can take place through the use of the special '%' character at the beginning and/or end of the search value. Note that when fields on the tracking tab are dimmed, it could indicate that the either the feature is not licensed or it is not enabled (refer to to Transfers under Configure System Options).

The […] buttons are present when Trading Partners are available for selection. These buttons allow the user to display the Trading Partners and configured Interchange Identifiers/Qualifiers. See Trading Partner Management

If a Trading Partner Alias is selected along with Use All Interchange Identifiers/Qualifiers, then a trading partner alias variable will be used. This will match any of the Interchange Identifiers/Qualifiers configured for the Trading Partner. If Use All Interchange Identifiers/Qualifiers is not selected, then the user can select a specific Interchange Identifier/Qualifier pair to be used. Once the selections have been made and OK is selected, then the selection will be placed in the appropriate fields depending on which button was selected.
