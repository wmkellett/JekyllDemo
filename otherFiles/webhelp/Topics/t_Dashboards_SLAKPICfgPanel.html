SLA/KPI Summary Configuration Panel

Use the configuration panel for the SLA/KPI Summary to control the data displayed in the summary.

Within the SLA/KPI Summary, you can modify the Start Date/Time and the End Date/Time, and the zoom slider on and off. See Filtering Dashboard Component Data for information on filtering data with zoom sliders. When toggled on, you can use the zoom slider to focus on the trading partner(s) of your choice.

See Cleo Dashboards and System Monitors Overview for general information about using these panels.
