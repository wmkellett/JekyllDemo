Resources Panel

The Resources panel is available when you click the Show Resources icon. Use this panel to drag resources onto the dashboard. 

The resources included in this panel are:
  • Component Library: Contains all available library components (*.lc files).
    • My Components: Contains all private library components. This is currently reserved for future use.
    • Public Components: Contains all public library components.
    • Business Dashboards: Contains all library components included in the baseline Cleo Dashboards™ product.
    • Custom Business Dashboards: Contains all custom library components that have been added for your specific application of the Cleo Dashboards™ product.
    • System Monitor: Contains all custom library components that have been added for your specific application of the Cleo System Monitor™ product.
    • Custom System Monitor: Contains all custom library components that have been added for your specific application of the Cleo System Monitor™ product.
  • Reports: Contains all available web reports (*.wls files).
    • My Reports: Contains all private web reports. This is currently reserved for future use.
    • Public Reports: Contains all public web reports.
    • Business Dashboards: Contains all web reports included in the baseline Cleo Dashboards™ product.
    • Custom Business Dashboards: Contains all custom web reports that have been added for your specific application of the Cleo Dashboards™ product.
    • System Monitor: Contains all web reports that have been added for your specific application of the Cleo System Monitor™ product.
    • Custom System Monitor: Contains all custom web reports that have been added for your specific application of the Cleo System Monitor™ product.
  • Toolbox: Contains all the tools that can be used in dashboard.
    • Dashboard Title: A special label that can be used in the dashboard header.
    • Image: An image that can be used in the dashboard header.
    • Label: A general label that be used in the dashboard header.
    • Special Field: A special field (date, user, etc.) that can be used in the dashboard header.
    • Slider: A special filter that can be used in the dashboard body.
    • Filter Control: A general filter that can be used in the dashboard body.
    • URL Frame: A frame through which you can provide access to a third-party web page, used in the dashboard body.
    • HTML: A general control for adding special messages and instructions to the dashboard body.
