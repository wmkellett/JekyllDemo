Launching Cleo System Monitor

Before you can access the Cleo System Monitor™ software, your system administrator will need to set up the appropriate access through the VersaLex and Cleo VLNavigator applications. After doing so, your administrator should provide an appropriate launch URL in the following format:

http(s)://VersaLexComputerIP:http(s)Port/SystemMonitorResourcePath
VesraLexComputerIP
IP address of the computer where the Cleo Harmony software is installed
http(s)Port
Port number where the Cleo Harmony software is listening for HTTP or HTTPs activity
SystemMonitorResourcePath
The path to your Cleo System Monitor resources. Configure this property in Local Listener Web Browser properties. See Configuring Dashboards and System Monitor for Web Browser service.
  1. Enter the URL provided by your administrator in your browser.
    A login dialog box displays.
  2. Enter your username and password and click Login. 
  3. The Cleo System Monitor module allows access to only one VersaLex at a time. If your user account is associated with multiple pools and multiple instances of Cleo Harmony software, you are prompted to choose a pool and then an instance of Cleo Harmony software within that pool. 
    The version of Cleo Harmony software in the pool you select is compared to the version of the report files in the report server. If the versions of the report files do not match the version of the instance you selected, then the report files will be published (see Publishing Reports). The browser displays a window containing some report information.
  4. If you need access to both Cleo Dashboards and Cleo System Monitor software at the same time in the same browser, you must create a new session for the second application and then use the appropriate URL.
    The steps to create a new session differ depending on which browser you are using.
    • For Internet Explorer, choose File > New Session.
    • For Chrome, choose File > New Incognito Window.
    • For Firefox, choose File > New Private Window.
