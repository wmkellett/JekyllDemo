Summary Dashboard Filter Area

The following filters are provided on the Summary Dashboard. These filters are associated with all library components on the dashboard. When you select one filter, all the components will be updated accordingly.

  • Trading Partner Filter: This filter control is bound to the Trading Partner field and controls all components on the dashboard.
  • Host Folder Filter: This filter control is bound to the Host Folder field and controls all components on the dashboard.

See Filtering Dashboard Component Data for more information about manipulating sliders and filter controls.The range of data values represented in the filters is comprised of all the values within the current record set, defined as all records returned from the most recent query. Since the Summary Dashboard is configured to automatically refresh, the database will be re-queried on each refresh. However, queries are also executed at these times:

  • When the dashboard is first launched.
  • When Refresh is clicked on the side bar.
  • When new parameters are submitted by clicking Enter Parameter Values on the side bar and entering new parameter values into the Enter Parameter Values dialog box. See Summary Dashboard Parameters.
