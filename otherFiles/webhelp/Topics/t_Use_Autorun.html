Use Autorun

As an alternative to remotely running Cleo LexiCom commands using the command line –s “remote,server” option, the autorun feature can be used to automatically process command files when they appear in a specific directory. Autorun commands are identical to the options available when executing Cleo LexiCom from the command line except for –m, –e, and –s (refer to Execute from Command Line).  In fact, the syntax of the command file matches the syntax of the –f “path” command line option. This feature is different from just executing LexiCom with command line options because the LexiCom commands can be generated from a different computer than one on which LexiCom is actually running.

The autorun feature does not need to be specifically enabled. As long as LexiCom is running (either the GUI or –s service), the feature is on. The default ‘autorun\’ directory can be modified (refer to Configure System Options).  No special naming convention is required for the command files. A command file is deleted when LexiCom finishes processing the commands.  Only one command file is processed at a time.
