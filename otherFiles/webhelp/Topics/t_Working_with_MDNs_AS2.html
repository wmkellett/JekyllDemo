Working with MDNs

An MDN is an acknowledgment sent in response to an AS2 message.

View Message Disposition Notifications (MDNs) received by remote hosts through the MDNs tab.

View the details of any particular MDN entry in the table by either right-clicking on a specific row and choosing Display from the pop-up menu, or by double-clicking on any row to display a detail window containing the contents of the MDN:

VersaLex appends tracking information to the MDN, including the Message Integrity Check (MIC) value it computed and the name of the file and the date and subject of the message that was sent. It also includes information about the sender and recipient of the message and how the message was assembled: for example, whether it was signed, encrypted, compressed, etc.

This information can be copied to the Windows clipboard by clicking the Copy button on the bottom of display window. The contents can then be pasted, for example into a new document or an email message. This information may also be printed to a networked printer by clicking the Print button on the bottom of the display window, or by selecting any one entry on the MDN display (shown in the previous diagram) and using the “Print” right-click menu option.
