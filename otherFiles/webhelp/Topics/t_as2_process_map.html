AS2 Process Map

If you are just getting started with VersaLex, refer to the Program Overview and VersaLex Basics. Then follow these steps to set up the Generic AS2 host:
  • Complete the AS2 Checklist
  •  AS2 Configuration
  •  Acquire Your Trading Partner's Signing/Encryption Certificates
  •  Determine/Provide Your URL Information
  •  Create/Provide Your Signing/Encryption Certificates
  • Complete configuration of:
    • Local Listener
    •  AS2 Host Configuration
    •  AS2 Mailbox Configuration
    • Compose an Action
    •  Testing Your AS2 Installation
