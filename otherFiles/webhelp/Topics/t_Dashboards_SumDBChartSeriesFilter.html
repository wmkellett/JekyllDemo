Changing Summary Dashboard chart, series, and filter

The Go to by Value options are similar to the Go To options; the only difference is that the Go to by Value options will filter the data, in addition to changing the category/series. For example, using the same chart as before (see Summary Dashboards Transfers Bar Chart), if the series has been reclassified by Host\Mailbox rather than General Status but the user wants to see the Failed types of transfers, follow this procedure:

  1. Right-click on the Failed legend label along the side of the chart. 
  2. Expand the Go to by Value submenu by hovering over the label.
  3. Click on the submenu item Host\Mailbox to render the chart with the new parameters. 

These steps demonstrate the Go to by Value options for changing and filtering the series; the steps are the same for changing and filtering the category.
