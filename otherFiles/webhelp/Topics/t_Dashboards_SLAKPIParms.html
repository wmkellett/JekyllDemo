SLA/KPI Dashboard Parameters

The SLA/KPI Dashboard contains the following configuration parameters:

  • Start Date/Time: this parameter specifies the starting date/time for the query. The factory-supplied default value is DateAdd(‘h’, -24, CurrentDateTime()); this translates to "24 hours prior to now".
  • End Date/Time: this parameter specifies the ending date/time for the query. The factory-supplied default value is CurrentDateTime(); this translates to "current date/time".
These factory-supplied parameter settings can be overridden by entering new parameter values. See Specifying Dashboard Parameter Values for more information about setting parameters.
Note: Due to practical memory limitations, if you specify a date range that produces a large number of records, the query will automatically limit the number of records. This limit is currently 100,000 records; however, this value is expected to increase in the future. If the date range specified creates a condition where the number of records has been limited, a warning message will be displayed at the bottom of the Transfer Details table. This indicates that the data set is incomplete. After receiving this warning, the date range should be reduced until the warning goes away.
