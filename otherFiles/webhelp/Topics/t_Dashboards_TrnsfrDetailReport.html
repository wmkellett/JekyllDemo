Transfer Detail Report

This report provides all the details of the selected transfer. If the transfer has associated tracked EDI data, a View Tracked EDI Data link will appear at the bottom of the report. If you click the link, it will display the Tracked EDI report. See Transfer Detail Report – EDI. Similarly, if the transfer has associated tracked XML data, a View Tracked XML Data link will appear at the bottom of the report; and, if the transfer has associated tracked TEXT data, a View Tracked TEXT Data link will appear. If the transfer is not associate with any tracked data, no links will appear.
