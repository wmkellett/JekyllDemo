Transfer Status Filter - VersaLexes Tab

You can filter by specifying a date range or any filter selections on the other tabs. If logging transfers to a database and synchronizing hosts across multiple instances of Cleo Harmony software, you can also filter based on which instance performed the transfer. By default, all synchronizing instances of Cleo Harmony software are included.  Also, if numbers have been specified in Options>Transfers>Additional Serial Numbers field, those numbers will be displayed. See Transfers. The All and None buttons allow you to select all or none of the instances of Cleo Harmony software.
