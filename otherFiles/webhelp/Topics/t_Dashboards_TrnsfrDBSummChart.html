Transfer Summary Chart

The Transfer Summary chart summarizes the transfers for each trading partner bound by the current filter (see Transfer Dashboard Filter Area) and parameter settings (see Transfer Dashboard Parameters). The initial chart categorization is Trading Partner and the chart series is General Status. These settings can be changed using the Go To and Go to by Value right-click menu options.
