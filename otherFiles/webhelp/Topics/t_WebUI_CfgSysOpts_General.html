Specifying Default Host Directories

  1. In the web UI, go to Administration > File System > Directories. In the native UI, select Configure > Options in the menu bar or click Options in the toolbar.
  2. Specify values for the fields on the page and then click Save.
    For information about fields and possible values, see Default Host Directory Reference.
