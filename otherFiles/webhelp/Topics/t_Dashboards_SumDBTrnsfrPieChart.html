Summary Dashboard Transfers Pie Chart

The Transfers pie chart summarizes the transfers for each trading partner bound by the current filter (see Summary Dashboard Filter Area) and parameter settings (see Summary Dashboard Parameters. The initial chart categorization is General Status;  however, this can be changed using the Go To and Go to by Value right-click menu options. See Changing the Summary Dashboard chart and series and Changing Summary Dashboard chart, series, and filter.
