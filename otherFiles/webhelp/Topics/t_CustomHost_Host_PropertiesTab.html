Connector Host: Properties Tab

The label of the Properties tab reflects the name of the connector. When you activate a connector, it has connector properties for which you assign values, some of which are required. This tab is where you assign these values.

Each property for each connector host includes tooltip-style documentation. In addition, you can press Help to display a dialog box containing documentation for all connector properties.

Note: When using the URI syntax to access this connector host, some connector properties might overlap with properties of the originating host (for example, Command Retries or Do Not Send Zero Length Files). When this occurs, which property is used (originating host or Connector) depends on the host making the decision. In the case of Do Not Send Zero Length Files, it is the originating host making the decision and that same property in the connector host is not used.
