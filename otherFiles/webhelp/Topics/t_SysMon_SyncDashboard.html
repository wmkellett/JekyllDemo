Synchronization Dashboard

The Synchronization Dashboard provides information related to the synchronization status of the currently selected VersaLex with other VersaLex systems.

Pool Information Table

The Pool Information table displays the pool name and the number of VersaLexes contained within the pool. These are typically static values, unless VersaLexes are being added or removed from the pool.

There are no sub-reports from this library component.

Synchronization Totals Table

The Synchronization Totals table displays counts of connection errors, unresolved items, and pending items to be synchronized with all VersaLexes in the pool. These counts are from the standpoint of the specific VersaLex being viewed.

It is normal for Pending Items to grow and shrink as the system is used. If the Pending Items grow abnormally large, that is typically an indication that another VersaLex is down. Any Unresolved Items indicate that manual intervention is required.

There are no sub-reports from this library component.

Synchronization Status Table

The Synchronization Status table displays the synchronization status and counts of the pending/unresolved items for each of the VersaLexes synchronized with the displayed VersaLex.

It is normal for Pending Items to grow and shrink as the system is used. If the Pending Items grow abnormally large, that is typically an indication that the specific VersaLex is down. Any Unresolved Items indicate manual intervention is required.

Access the Synchronization History sub-report by clicking on the underlined Serial Number in the table. Access the Unresolved Synchronization Items sub-report by clicking on the underlined value in the table.

Synchronization History Report

The Synchronization History report displays the number of pending and unresolved items over time. By default, it will show the last 60 minutes of history. Use the parameter control box to change the Start date/time of the chart and the number of Minutes displayed. The zoom slider at the bottom can zoom in closer to a specific time. Hover the cursor over a specific data point on the chart to display information about that data point.

Unresolved Synchronization Items Report

The Unresolved Synchronization Items report displays a table of items for which the synchronization will need manual correction through VersaLex.
