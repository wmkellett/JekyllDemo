Publishing Reports

Each time a user logs into the Cleo Dashboards™ or Cleo System Monitor™ software, the current VersaLex version is compared against the version of published files in the report server. If the VersaLex version is newer, the report files stored in the VersaLex BI sub-folder are published to the report server. During this time a status shows the publishing progress.

After the report files are published, the baseline dashboards are displayed. 

If any users were viewing dashboards and the report files were being updated, an error could occur and a message is displayed.

The user must log out and log back in to clear the error.

If the report files published on the report server are from a version of VersaLex newer than the VersaLex that you are accessing the dashboards from, an error occurs and a message is displayed.

To access the dashboards, you will either need to log in through the updated VersaLex server or update the VersaLex server to the same version of software.
