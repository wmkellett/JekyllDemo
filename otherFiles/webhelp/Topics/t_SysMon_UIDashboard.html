UI Dashboard

The UI (User Interface) Dashboard provides information related to the users logged into the currently selected VersaLex. 

Connected VersaLex/VLNavigator Users Chart

The Connected VersaLex/VLNavigator Users chart displays a count of the number of logged in users in four categories: VersaLex Native Users, VersaLex Web Users, VLNavigator Native Users, and VLNavigator Web Users.

You can monitor how many users are accessing the system in the various modes and compare against normal levels.

Access the User Interface History sub-report by clicking on a bar or a bar label.

User Interface History Report

The Unsolicited Status History report displays the number of users logged into the various products over time. By default, it will show the last 60 minutes of history. The parameter control box can be used to change the Start date/time of the chart and the number of Minutes displayed. Use the zoom slider at the bottom to zoom in closer to a specific time. Hovering the cursor over a specific data point on the chart will display information about that data point.

Active Users Table

The Active Users table displays a count of the total number of VersaLex and VLNavigator users currently logged in.

Access the Current Users sub-report by clicking on the underlined value in the table.

Current Users Report

The Current Users report displays a table of the users logged into the various products, where they are logged in from, and when they logged in.
