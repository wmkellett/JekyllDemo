Native File System Installation

If this is the first time you have installed on the AS/400 or you are installing a release and not a patch, you must follow the procedure in this step to complete the LexiCom NFS installation on the AS/400.  If you are not installing a major release, you may skip this step.

 

  1. Verify that the "LexiCom.savf" file has been copied (or unzipped from the AS400.zip file) to the /LexiCom folder in the IFS.
  2. Sign on to the AS/400 "green screen" as QSECOFR and type the following command:

    CPYFRMSTMF FROMSTMF('/LexiCom/LEXICOM.savf') TOMBR('/QSYS.LIB/QGPL.LIB/LEXICOM.file')  CVTDTA(*NONE)

    (Verify that the message: Stream file copied to object is displayed.)

  3. Restore all the objects required to complete the installation:

    RSTOBJ OBJ(*ALL) SAVLIB(QGPL) DEV(*SAVF) SAVF(QGPL/LEXICOM)

    (Verify that the message:  7 objects restored from QGPL to QGPL is displayed.)

  4. Type the following command to install all the necessary LexiCom commands and objects on the AS/400:

    CALL INSTLEX

    (Verify that the message: LexiCom AS/400 Installation Complete is displayed.)
