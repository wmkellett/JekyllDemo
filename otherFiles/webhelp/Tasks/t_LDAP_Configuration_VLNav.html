LDAP Configuration

This configuration is optional unless you intend to define LDAP User(s) who authenticate with an external directory service, such as Active Directory or Apache Directory Service, instead of VLNavigator. 

  1. Select Users in the tree pane and select the LDAP Server tab.
  2. Enter the required user information. See Users LDAP Server for a description of this tab.
  3. Click Apply.
