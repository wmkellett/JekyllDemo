AS/400 Install

To install and run the AS/400 version of Cleo LexiCom software, a portion of the software will be installed using a Windows PC mapped to the Integrated File System (IFS) and then a portion will be installed in the AS/400 Native File System through "green screen" commands.   After all the requirements described in the previous section have been satisfied, use the following two sections to complete the installation to the Integrated and Native File Systems. 

Note: The following procedure should only be completed in its entirety the first time you install LexiCom.  If you are upgrading to a new version of Cleo LexiCom software, first verify the product is not running on the AS/400 by either typing the ENDLEXSVR CL command on the "green screen" or by selecting the "Stop AS/400" option from within the Cleo LexiCom software.

and then obtain the latest software updates. See Integrated File System Installation for information.
